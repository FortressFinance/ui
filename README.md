# Fortress Finance UI

## Requirements

- Node 16
- Yarn

## Getting Started

1. Install dependencies

```sh
$ yarn
```

2. Create a `.env` file based on `.env.example`. You can ask another dev for the
   environment variables or fallback to public RPC providers.

```sh
$ cp .env.example .env
```

3. Start the dev server

```sh
$ yarn dev
```
